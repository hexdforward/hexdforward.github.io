一些写在刚开始的话。

起源

很早就有了输出一些东西的想法，大概16年4月看到当时还在CMU读书的小土刀写的CSAPP系列博客，以及随后看的博客中的一些其他的内容，从他的博客中能看出他是一个自律、勤奋、优秀的人，每次在消沉期翻看博客都能获取一波正能量，我也想成为他这样优秀的人。所以显而易见，我也希望自己在前进的过程中多一些思考和积累，希望自己变得优秀，所以有了这个博客。

内容

我暂时将博客的内容分为以下几类：调与研、架与构、读与记

  • 调与研：在调研新知识和新模块的问题和解决问题过程的积累，积累的过程对于理解问题以及之后再次回想时有很大帮助
  • 架与构：完整知识体系的总结，方便记忆，遇到知识关联时有助于快速回想
  • 读与记：读书后的思考、感想、记录

更新

初尝试写博客，平时说话时并不觉得无话可说，可是真正提笔写时才发现囊中羞涩，所以在更新这一块暂时不给自己设要求，有时间就不定期更新，内容质量不限。
